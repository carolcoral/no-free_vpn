# vpn

部分免费的 VPN。亲测有效的科学上网，同时支持 windows、mac、linux、ios 和 andrioid 系统。并提供 chrome、firefox、opera 等浏览器的插件使用。

**推荐使用 [SS](https://carolcoral.github.io/no-free_vpn/shadowsocks)**
